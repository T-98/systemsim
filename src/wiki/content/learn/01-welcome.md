# Welcome to SystemSim

SystemSim is a distributed-systems design simulator. You describe a system, the canvas assembles it, and the engine runs traffic through it until something breaks. You learn by watching real numbers — utilization, p99 latency, error rates, queue depth — not by memorizing definitions.

## Three ways to start

- **From a template.** Click a starter on the landing page. Fast path to a running sim.
- **From plain English.** Type a description ("Instagram-like feed, 10M DAU, photos and likes"). The AI picks components and lays them out.
- **From an image.** Paste or upload a diagram (Miro, Figma, Excalidraw, even a handwritten sketch). The AI reads the shapes and builds the canvas.

## What you'll learn here

Every feature has a page in this Learn track. The Reference tab covers the system-design concepts themselves — caching, sharding, consistency, resilience — in long-form. The How-to tab has five scenarios that reproduce common failure modes you can break and fix.

Pick up where you want. The order in the sidebar is the intended reading order, but each page stands alone.
